# PyScan
## About
### This app is simply app for scan documents and images.
### App using:
1. PyQt6
2. python-sane
3. tomlkit
4. PIL modules.
### Why PyScan?
1. Simply
2. Customizible
3. Clear
## How to build
### 1. Clone repo:
    git clone https://github.com/whyDEDUS/PyScan.git
### 2. Start build:
    python3 setup.py install
